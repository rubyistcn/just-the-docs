# 标签

用标签做为给文档章节添加附加标记的一种方式。标签可以有一些颜色。默认情况下，标签是蓝色的。

<div class="code-example" markdown="1">
Default label
{: .label }

Blue label
{: .label .label-blue }

Stable
{: .label .label-green }

New release
{: .label .label-purple }

Coming soon
{: .label .label-yellow }

Deprecated
{: .label .label-red }
</div>
